# higher-lower-game

A simple command-line "Higher or Lower" game written in Python. The player is shown two choices and must guess which one has a higher value (for example, follower count). It's a small, educational project for practicing Python basics and control flow.

Features

- Two-option comparison rounds
- Score-keeping across rounds
- Simple, dependency-free Python code

Getting started

1. Ensure you have Python 3.8+ installed.
2. Run the game:

```bash
python main.py
```

Project layout

- `art.py` — ASCII art and visual assets
- `game_data.py` — dataset used by the game
- `main.py` — game entry point
- `README.md` — project documentation
